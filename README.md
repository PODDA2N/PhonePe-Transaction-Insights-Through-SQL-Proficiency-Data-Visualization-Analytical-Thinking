📊 PhonePe Transaction Data Insights (hashtag#SQLProficiency, hashtag#DataVisualization also Using hashtag#StreamlitApp to present the analysis results)

📌 Project Overview
This project focuses on analyzing PhonePe transaction data using SQL. The dataset includes structured information across three major table categories:

🔹Aggregated Tables
🔹Map Tables
🔹Top Tables
The goal is to extract meaningful insights related to transaction volume, user growth, and payment behavior.

✅ Data Extraction:
Collected and processed structured transaction data from CSV files, ensuring proper formatting and integration into a MySQL database for efficient querying.

✅ SQL Proficiency

📁 Table Categories
1️⃣ Aggregated Tables

Contains summarized transaction data:
Used for high-level insights such as total transactions and total amount
Example metrics:
Total transaction count
Total transaction amount.

2️⃣ Map Tables

Provides geographical insights:
Data is categorized by states and districts
Useful for:
Regional transaction analysis
User distribution mapping.

3️⃣ Top Tables

Highlights top-performing entities:
Includes rankings based on:
Highest transaction count
Highest transaction amount
Most registered users

✅ Exploratory Data Analysis (EDA)
Cleaned and transformed raw data using Python (Pandas, NumPy)
Conducted univariate, bivariate, and correlation analysis to understand patterns. Utilize Python libraries (e.g., Pandas, Matplotlib, Seaborn) to analyze the results from the SQL queries.

✅ Data Visualization
Created insightful visualizations using Matplotlib & Seaborn
Highlighted trends like transaction distribution, brand dominance, and state-wise performance.

✅ Interactive Dashboard (Streamlit)
Developed a dynamic dashboard for real-time data exploration
Implemented filters, KPIs, and charts for better user experience.

🔹 Analytical Thinking:
Identified key insights such as regional variations in digital payment adoption, correlation between transaction count and amount, and dominant players in the market. Translated raw data into actionable business insights.

Overall, this project demonstrates the ability to transform raw transactional data into data-driven insights and interactive dashboards, showcasing strong skills in SQL, Python, and data analysis.

📌 Key Insights
UPI-based transactions dominate across all regions
Certain states contribute significantly to total transaction volume
Rapid growth observed in registered users over time
Top tables reveal concentration of transactions in major regions.
